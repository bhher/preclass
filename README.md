# 풀스택 선행수업(9시간) 안내 페이지

## 실행 방법

- `preclass/index.html` 파일을 브라우저로 열면 됩니다.
- VS Code에서 더 편하게 보려면(권장):
  - 확장(Extensions)에서 **Live Server** 설치
  - `index.html` 우클릭 → **Open with Live Server**

## 파일 구성

- `preclass/index.html`: 메뉴/탭이 있는 본문
- `preclass/styles.css`: 스타일(라이트/다크 자동)
- `preclass/app.js`: 탭 전환 로직(키보드/접근성 포함)
